## Learning React 101

React.js starter kit with Babel and Gulpfile.

## Setup

To setup the package.

```
npm install --global gulp
npm install
gulp build
```

To watch file changes.

```
gulp watch
```

Browser live reload.

```
gulp browser
```

## Usage

Edit ```src/hello.jsx``` file and run ```gulp build```, the compiled file is at ```dist/hello.js```.
